Create custom blog info fields that can be used sitewide in Elementor.

== Description ==

Blog Info allows you to create and manage custom meta fields that can be used throughout your site in Elementor. This is perfect for storing and displaying information like:

* Brand name and blog niche
* Social media links
* And more!

Once you've set up your meta fields, you can use them in any Elementor widget that supports dynamic tags.

= Features =

* Easy-to-use admin interface for managing blog info fields
* Support for different field types (text, url)
* Elementor dynamic tags integration
* Use your blog info fields in any Elementor widget that supports dynamic content

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/elementor-sitewide-meta` directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to 'Blog Info' in your WordPress admin menu to manage your fields.
4. Use the dynamic tags in Elementor to display your blog info fields.

== Frequently Asked Questions ==

= Does this plugin require Elementor? =

Yes, this plugin requires Elementor to be installed and activated.

= How do I use the blog info fields in Elementor? =

1. Edit a page or post with Elementor.
2. Select a widget that supports dynamic content (like Heading, Text Editor, etc.).
3. Click the dynamic content icon (database icon).
4. Select "Blog Info Field" from the list.
5. Choose the field you want to display.

= Can I reset all field values? =

Yes, there's a "Reset All Values" button at the bottom of the Blog Info page that will clear all field values while keeping the field structure intact.
